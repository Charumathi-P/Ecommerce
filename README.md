# 🌌 AETHER – Premium Tech E-Commerce Website
AETHER is a modern and responsive e-commerce website designed for premium smart gadgets and lifestyle accessories. It delivers an elegant shopping experience with a clean UI, dark/light theme support, wishlist, shopping cart, and responsive layouts.

---

## 🚀 Features
- 🏠 Modern Landing Page
- 🛍️ Product Catalog
- 🔍 Product Search & Categories
- ❤️ Wishlist
- 🛒 Shopping Cart
- 🌙 Dark & Light Theme
- 📱 Fully Responsive Design
- 📧 Newsletter Subscription
- 📄 About & Contact Pages
- 🎨 Premium Minimal UI/UX

---
